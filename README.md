# 2. MyDriverBook – ELD & HOS Compliance System (USA Trucking Industry)

Project Description
MyDriverBook is an **ELD-based fleet compliance system** designed for truck drivers in the United States. The application helps commercial drivers comply with **FMCSA Hours of Service (HOS) regulations** by automatically tracking driving activity, vehicle inspections, and driver logs through Electronic Logging Devices.

My Role & Responsibilities

* Built the mobile application using React Native for both Android and iOS
* Integrated **ELD SDKs** to communicate with vehicle logging devices in real time
* Implemented **HOS tracking logic aligned with FMCSA regulations**
* Built background services to support continuous GPS tracking and driving activity monitoring
* Developed **DVIR (Driver Vehicle Inspection Report)** module with digital signature capture
* Implemented hybrid **ELD + GPS fleet tracking system** for real-time monitoring
* Maintained and upgraded React Native dependencies for improved cross-platform stability

Technologies Used
React Native, TypeScript, ELD SDK Integration, GPS Tracking, Background Services, Zustand, Background Service, REST APIs, Git

Outcome
Enabled **automated driver log recording and compliance reporting**, reducing manual entry errors and helping fleets maintain regulatory compliance.
